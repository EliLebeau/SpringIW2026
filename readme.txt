Many tutorials from W3 schools and Geeks for Geeks were helpful in string processing and using various Python packages for performing the involved analysis. The SentenceTransformers Documentation was also of use in embedding and calculating the cosine similarity of verses.

Hebrew_Lemmatization.ipynb: notebook for lemmatizing Hebrew verses
LatinScraper.py: Python module for extracting Latin text. Greek and Hebrew extraction not included. Greek text was processed from an xml file. Hebrew Text was extracted manually
claudeGreek.py: Python module for submitting csv file of each psalm from the Septuagint to claude for translation
claudeHebrew.py: Python module for submitting csv file of each psalm from the Hebrew Bible to claude for translation
claudeLatin.py: Python module for submitting csv file of each psalm from the Gallican Psalter and the Hebrew Psalter to claude for translation
final_labse_result.csv: results of labse embeddings and cosine comparisons
